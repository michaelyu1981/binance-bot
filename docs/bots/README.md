# Bot Profiles

This folder stores advisory bot personality and strategy profile notes.

Examples:

- `bot_michael_burry.md`
- `bot_ed_seykota.md`

Rules:

- These files are documentation only.
- They must not contain secrets.
- They must not contain Binance API keys, Telegram tokens, passwords, or server
  credentials.
- They must not enable trading.
- They must not bypass project safety gates.
- Any future code that uses these profiles must stay read-only or dry-run unless
  Michael explicitly approves the required phase.
